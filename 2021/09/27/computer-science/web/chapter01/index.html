# HTML 基本标签

  • <!DOCTYPE> : 位于文档的最前面，用于向浏览器说明当前文档使用哪种 HTML 或 XHTML 标准规范。
  • <html>...</html> : 告知浏览器是一个 HTML 文档。
  • <head>...</head> : 定义文档的头部信息。
  • <body>...</body> : 主体标签，定义文档要显示的内容。
  • <title></title> : 定义 HTML 页面的标题。
  • <mate/> : 定义页面的元信息，可重复出现。
  • <link> : 引用外部文件，可用多个。
    • href
      • URL: 指定外部文档位置。
    • rel
      • stylesheet: 指定当前文档与引用外部文档的关系，该属性值通常为 stylesheet ，表示定义一个外部样式表
    • type:
      • text/css: 引用外部文档的类型为 CSS 样式表
      • text/javascript: 引用外部文档的类型为 javascript 脚本
  • <style>...</style> : 定义样式信息。
  • <hn>...</hn> :n 取 16，依次表示 16 级标题。
    • align
      • left
      • center
      • right
  • <p>...</p> : 段落标签。
  • <hr/> : 水平线
    • align
    • size: 粗细
    • color: 颜色
    • width: 宽度
  • <br/> : 强制换行。
  • <font>...</font> : 文本样式标签
    • face: 字体
    • size: 大小，1~7
    • color: 颜色
  • <b></b> 、 <strong></strong> : 文字粗体。
  • <i></i> 、 <em></em> : 文字斜体
  • <s></s> 、 <del></del> : 文字删除线
  • <u></u> 、 <ins></ins> : 文字下划线
  • 特殊字符

  • 无序列表
  • 有序列表
  • 定义列表
  • <img/> : 图像标签
    • src: 指定图像路径
    • alt: 图像的替换文本属性
    • width: 图像宽度
    • height: 图像高度
    • border：添加边框，设置边框的宽度
  • 绝对路径：计算机上的磁盘位置，带盘符或网址。
  • 相对路径：与文档目录相同。
  • <a></a> : 超链接
    • href: 跳转目标
    • target
      • _self: 在原窗口打开
      • _blank: 在新窗口打开
  • 锚点连接

# 框架

iframe 内联框架
在网页中打开的一个小窗口，可以引入其他网页
属性：

  • src: 引用页面地址
  • name：框架标识名
    * frameborder：边框
    * width, height：框架宽高
    * scrolling (yes/no)：是否出现滚动条
    * noresize：是否能调整大小

# 一、表单的基本语法

  • form：表单标签
    内含表单元素，所有要向服务器提交的数据都必须放入表单标签中。

  • 属性：
    method(get/post) ：提交方式

    • get ：参数在地址栏可见，不安全；长度有限制 (255)
    • post ：参数在地址栏不可见，安全；长度不限

    action ：服务器地址（将表单中的数据提交给服务器）
    name ：表单的内部引用名

# 二、表单元素

  1. 文本框（text）: 用于用户输入字符信息

    • 属性：
      value ：表示用户输入的值或者初始值。
      size ：表示文本框长度
      maxlength ：表示可接受的字符数

    • 效果：

  2. 密码框（password）：用于用户输入密码信息

    • 属性：
      value ：表示用户输入的密码或者初始密码。
      size ：表示文本框长度
      maxlength ：表示可接受的字符数

    • 效果：

  3. 单选按钮（radio）: 便于用户选择单个信息

    • 属性：
      单选按钮通常是一组，同一组的单选按钮名称必须相同，但值要不同，这样在选中一个按钮时，其他按钮会自动取消
      name ：单选按钮的名字，同一组的名称相同
      value ：选中本按钮时，提交的值
      checked ：默认选中本按钮

    • 效果：

  4. 复选框（checkbox）: 便于用户选择多个信息

    • 属性：
      复选框通常是一组，同一组的单选按钮名称必须相同，但值要不同，复选框可同时选择多个
      name ：复选框的名字，同一组的名称相同
      value ：选中本按钮时，提交的值
      checked ：默认选中本按钮

    • 效果

  5. 文件域（file）: 用于用户提交文件
    文件域通常显示为一个文本框和一个文件选择按钮
    name ：文件域的名字
    若想提交文件，必须为 form 表单添加属性： method="post", enctype="multipart/form-data"

    效果：

  6. 提交按钮（submit）: 用于提交表单中的数据，点击提交按的默认动作就是提交数据。

    • 属性：
      value ：按钮上显示的文字
      name ：按钮的名字，通常没什么用，省略

    • 效果

  7. 重置按钮（reset）：用于提交表单中的表单元素恢复到初始状态。

    • 属性：
      value ：按钮上显示的文字
      name ：按钮的名字，通常没什么用，省略

    • 效果：

  8. 图片提交按钮（image）：和 submit 按钮的作用相同，只不过显示为一张图片.

    • 属性：
      src : 引用图片的位置
      name ：按钮的名字，通常没什么用，省略

    • 效果：

  9. 普通按钮（button）: 没有默认动作的按钮，可以添加自定义点击事件

    • 属性：
      value : 按钮文字
      name ：按钮的名字
    • 效果：

  10. 隐藏域（hidden） : 用于在页面上隐藏一些信息，这些信息不显示出来，但是会提交到服务器.

    • 属性:
      name ：隐藏域名称
      value : 隐藏域提交的值

  11. 下拉列表 (select-option): 提供给用户多个选项，供用户选择

    • select 属性：
      name ：下拉列表名称
      size : 下拉列表显示的个数，如果没有内容，则是空白。
    • option 标签说明一个选项：标签内容是显示的文字

      • 属性：
        value ：表示选择本选项时提交的值
        selected ：默认选中本选项

# 三、HTML5 新增表单元素

  1. Email 标签（email）: 与文本框功能类似
    自动进行邮箱格式验证。

  2. URL 标签（url）: 自动进行 URL 地址格式验证。

  3. number 标签 (number)： 自动进行数字验证
    设定数字 最大值(max) , 最小值(min) ， 步长(step)

  4. 数字范围标签 (range): 自动进行数字验证，通过滑块设置数字
    设定数字最大值 (max), 最小值 (min)，步长 (step)

  5. 日历标签 (date)： 日历选择控件，类型：month,week,time,datetime,datetime-local
    可设置日期的默认值

  6. 时间标签（time）：选择时间，几点几分，可设置时间的默认值。

  7. 搜索框标签 (search)： 用于设定搜索关键字
    与文本框的区别在于输入后自动显示关闭按钮，点击可删除关键字，可设置默认值。

  8. 颜色标签（color）：选择颜色。

# 四、表单元素的特殊属性

属性名含义适用范围
disabled禁用表单元素适用于各类标签
readonly只读适用于输入标签:text,textarea,password
checked默认选中某选项radio,checkbox
selected默认选中某选项select
style="resize:none"锁定文本域的大小textarea
required表示该项为必填项适用于输入标签:text,textarea,password
pattern对输入的内容进行自定义正则表达式验证适用于输入标签:text,textarea,password
placeholder提供可描述输入字段预期值的提示信息text, search, url, telephone, email 以及 password。

# 六、关于正则表达式

虽然 HTML 中提供了对某些输入内容进行验证的表单元素，但在实际应用中，我们还应根据实际情况自定义验证格式，这就需要用到正则表达式对输入内容进行验证，这里简单介绍一些正则表达式：

  • 正则表达式间的运算：
    • + : 拼接字符，可省略。
    • 或 [] ：在给定区间随便取一个数。
    • 幂 {} : 取规定数量的字符

\d : 表示匹配 0~9 的任何数字，等价于 [0-9]
\w ：表示匹配包括下划线的任何单词字符。等价于 '[^A-Z a-z 0-9_]
. : 匹配除 “\n” 之外的任何单个字符。

如我们要对输入的手机号进行验证，电话号码有 11 位，第一位必须是 1，第二位比较常见的是 356789，其他数字随意，那么对于的正则表达式可写为： 1[356789]\d{9}
