# 盒子模型

什么是盒子模型？
盒子模型
CSS 将网页看做是由一个一个的盒子 (box) 组成
主要的盒子属性有 (由外到内)： margin (外边距), border (边框), padding (内边距), width (宽)/ height (高)

# 边框

属性名含义使用取值
border-color设置边框颜色border-top-color : 上边框颜色 <br> border-right-color : 右边框颜色 <br> border-bottom-color : 下边框颜色 <br> border-left-color : 左边框颜色 <br> 简写： border-color :border-top-color,border-right-color,border-bottom-color,border-left-color<br>(即：上右下左，顺时针方向；另外，上和下，左和右对应，如果上下或者左右颜色相同，写一边即可)color
border-width设置边框粗细border-top-width : 上边框粗细 <br> border-right-width : 右边框粗细 <br> border-bottom-width : 下边框粗细 <br> border-left-width : 左边框粗细 <br> 简写： border-width :border-top-width,border-right-width,border-bottom-width,border-left-width<br>(即：上右下左，顺时针方向；另外，上和下，左和右对应，如果上下或者左右粗细相同，写一边即可)<length> ： 用长度值来定义边框的厚度。不允许负值 <br> medium ： 定义默认厚度的边框。 <br> thin ： 定义比默认厚度细的边框。<br> thick ： 定义比默认厚度粗的边框。
border-style设置边框风格border-top-style : 上边框风格 <br> border-right-style : 右边框风格 <br> border-bottom-style : 下边框风格 <br> border-left-style : 左边框风格 <br> 简写： border-style :border-top-style,border-right-style,border-bottom-style,border-left-style<br>(即：上右下左，顺时针方向；另外，上和下，左和右对应，如果上下或者左右风格相同，写一边即可)none ： 无轮廓。border-color 与 border-width 将被忽略 <br> hidden ： 隐藏边框。IE7 及以下尚不支 <br> dotted ： 点状轮廓。IE6 下显示为 dashed 效果 <br> dashed ： 虚线轮廓。 <br> solid ： 实线轮廓 <br> double ： 双线轮廓。两条单线与其间隔的和等于指定的 border-width 值
border-image图片边框border-image-source : 指定图片路径 <br> border-image-slice : 指定四边偏移量 <br> border-image-width : 指定边框宽度 <br> border-image-outset : 指定边框背景向盒子外部延伸的距离 <br> border-image-repeat :<br> 指定平铺方式

border 简写：同时设置边框的颜色、粗细和风格； border:border-width border-style border-color

# 内边距与外边距

边距含义使用取值
margin外边距magin-top : 边框上边框与上面边框的距离 <br> magin-right : 边框右边框与右面边框的距离 <br> magin-bottom : 边框下边框与下面边框的距离 <br> magin-left : 边框左边框与左面边框的距离<length>
padding内边距padding-top : 内容与上边框的距离 <br> magin-right : 内容与右边框的距离 <br> magin-bottom : 内容与下边框的距离 <br> magin-left : 内容与左边框的距离<length>
  • margin 简写：顺序上右下左，另外，上和下，左和右对应，如果上下或者左右外边距相同，写一边即可
  • padding 简写：顺序上右下左，另外，上和下，左和右对应，如果上下或者左右内边距相同，写一边即可
  • 网页居中对齐： margin:0px auto;

# 盒子模型的尺寸

盒子尺寸取值
box-sizingcontent-box ： padding 和 border 不被包含在定义的 width 和 height 之内。对象的实际宽度等于设置的 width 值和 border、padding 之和，即 ( Element width = width + border + padding ) 此属性表现为标准模式下的盒模型。<br> border-box ： padding 和 border 被包含在定义的 width 和 height 之内。对象的实际宽度就等于设置的 width 值，即使定义有 border 和 padding 也不会改变对象的实际宽度，即 ( Element width = width ) 此属性表现为怪异模式下的盒模型。

# 圆角边框

  • border-redius :Hlength1/Vlength1  Hlength2/Vlength2  Hlength3/Vlength3  Hlength4/Vlength4
    水平半径 (Hlength)：如果提供全部四个参数值，将按上左 (top-left)、上右 (top-right)、下右 (bottom-right)、下左 (bottom-left) 的顺序作用于四个角。
    如果只提供一个，将用于全部的于四个角。
    如果提供两个，第一个用于上左 (top-left)、下右 (bottom-right)，第二个用于上右 (top-right)、下左 (bottom-left)。
    如果提供三个，第一个用于上左 (top-left)，第二个用于上右 (top-right)、下左 (bottom-left)，第三个用于下右 (bottom-right)。
    垂直半径 (Vlength) 也遵循以上 4 点。
  • 也可以用百分比设置圆角半径

# 盒子阴影

  • box-shadow ：inset   length1   length2   length3   length4   color
    • inset : 设置阴影类型，默认为 none 无阴影，inset 为内阴影，若不填则为外阴影。
    • length1：X 轴位移
    • length2：Y 轴位移
    • length3：阴影模糊值
    • length4：阴影外延值
    • color：阴影颜色

# 背景属性

  1. background-color
  2. background-image
    • background-repeat
      • repeat : 沿水平和垂直两个方向平铺
      • no-repeat : 不平铺
      • repeat-x : 只沿水平方向平铺
      • repeat-y : 只沿垂直方向平铺
    • background-position
      • 数值：px
      • 关键字
        • 水平：left、center、right
        • 垂直：top、center、bottom
      • 百分比
    • background-attachment
      • scroll : 图像随页面元素滚动
      • fixed : 图像固定
    • background-size
      • 像素值
      • 百分比
      • cover
      • contain
    • background-origin ：定义背景图像的相对位置
      • padding-box : 相对于内边距定位
      • border-box : 相对于边框定位
      • content-box : 相对于内容框定位
    • background-clip : 定义背景图像裁剪区域
      • border-box : 从边框区域向外裁剪背景
      • padding-box : 从内边距区域向外裁剪背景
      • content-box : 从内容区域向外裁剪背景

# 元素类型转换

  • 块元素
    • 可以设置宽、高、对齐等属性。
    • <div>
  • 行内元素
    • 依靠字体大小和图像尺寸支撑结构
    • 不可以设置宽、高、对齐等属性。
    • <span> 、 <a> 、 <em>
  • 行内块元素
    • 依靠字体大小和图像尺寸支撑结构
    • 可以设置宽、高、对齐等属性。
    • <img> <input>
  • display
    • inline: 行内元素
    • block: 块元素
    • inline-block: 行内块元素
    • none: 被隐藏，不显示也不占用页面空间。
